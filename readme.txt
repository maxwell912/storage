Распределенное хранилище
Все модули можно импортировать и использовать реализованный функционал в коде.

Storage
Локальное хранилище данных в модуле. Имеет функции добавления, удаления данных по ключу.

Server.py
Файл запуска сервера. Ваш компьютер становится сервером с заданным IP, принимает от клиента данные в текстовом формате,
хранит их и выдает по требованию.

Client.py
На вход получает IP серверов-хранилищ. Отправляет запрос на хренение данных <ключ, значение> на несколько серверов
(кол-во задается в параметрах).
